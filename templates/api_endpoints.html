API Endpoints

POST /api/submitData/
Добавить новый перевал. Требует полного набора данных о перевале, включая координаты, уровень сложности и изображения.
GET /api/submitData/<id>/
Получить данные о конкретном перевале по его ID.
PATCH /api/submitData/<id>/
Обновить данные перевала (только если статус "new"). Нельзя изменять данные пользователя.
GET /api/submitData/?user__email=<email>
Получить список всех перевалов, добавленных пользователем с указанным email.

Для полной документации с возможностью тестирования запросов используйте Swagger UI или ReDoc.
